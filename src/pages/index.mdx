<Title>Documentation</Title>

<Boxes>
<Box title="Quick start guide" to="/getstarted">

Start with the basic steps of creating a Toit account, setting up your ESP32 with
the Toit firmware, deploying a Toit app on your device, and viewing the logs.

</Box>
<Box title="Tutorials" to="/tutorials">

Browse through the tutorials to learn how devices talk to each other in Toit,
and see how to use the different communication mechanisms on your ESP32, such as MQTT and PubSub.

</Box>
<Box title="Platform" to="/platform">

Learn about the concepts around the Toit platform, the device life-cycle as well as the applications that are deployed on Toit devices.

</Box>
<Box title="Firmware" to="/firmware">

Get an overview of the connectivity options ready-to-use and available with the Toit firmware, and which ESP32 models are supported.

</Box>
<Box title="Peripherals" to="/peripherals">

Explore how to work with GPIO in Toit as well as the various peripherals that Toit supports.

</Box>
<Box title="Language" to="/language">

Take the language tour and learn the Toit programming language tailored for embedded devices.
Learn how to adapt Toit packages in your code by utilizing the Toit package registry.

</Box>
<Box title="Cloud APIs" to="/apis">

Learn how to pull device data via the external Toit APIs into your own IoT cloud or third-party client application.

</Box>
<Box title="Support" to="/support/troubleshoot">

Support contains a troubleshooting guide, but also links to get in touch with the Toit engineers, if you have any questions or need help.

</Box>
</Boxes>
