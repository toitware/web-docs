# Download

<Boxes>
<Box title="Toit CLI">

Download the version of the Toit CLI that matches your current operating system.

Choose [Windows](https://archive.toit.io/cli/latest/windows/installer), [macOS](https://archive.toit.io/cli/latest/darwin) or [Linux](https://archive.toit.io/cli/latest/linux)

See [installation instructions](../installation).

</Box>

<Box title="Toit examples">

Download the Toit examples for the latest SDK.

This is an optional download.

Choose [ZIP](https://archive.toit.io/sdk/latest/examples/windows) or [TGZ](https://archive.toit.io/sdk/latest/examples/linux)

</Box>

<Box title="Toit SDK library">

Download the Toit library implementation for the latest SDK.

This download allows users to see the implementation of the core libraries, and is an optional download.

Choose [ZIP](https://archive.toit.io/sdk/latest/lib/windows) or [TGZ](https://archive.toit.io/sdk/latest/lib/linux)

</Box>
</Boxes>
