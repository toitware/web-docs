# SDK

The Toit SDK is the set of development tools needed to develop applications with Toit locally. It consists of the compiler, virtual machine, and
standard libraries that together enable Toit programs to run on an ESP32 or on your host machine.

The Toit SDK open source and available on GitHub. Find it [here](https://github.com/toitlang/toit).

You can browse the current Toit standard libraries [here](https://libs.toit.io).
