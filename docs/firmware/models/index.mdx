# Models overview

Models are designed to target different core hardware, allowing the Toit stack to be used across many different systems.

Today the primary model used is `esp32-4mb`.

## Model parameters

Some models supports custom model-parameters, e.g. to wire up core peripherals.

### Pin mode

When utility pins are exposed through model parameters, an optional mode may be given as well. Pins has 3 modes available as described below.

Value | Description | Notes
--- | --- | ---
0 | Configures the pin as active-low
1 | Configures the pin as active-high
2 | Configures the pin as open-drain active-low | **Default**
