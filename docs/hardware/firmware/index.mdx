# Toit firmware

The Toit Firmware is the Toit runtime system that is installed on your hardware, on top of the OS, to enable program execution.

In the [GPIO](gpio) section you learn how to use the General Purpose Input/Output pins with Toit.

Get an overview of the [device models](models) currently present in the Toit firmware.

The available protocols are described in detail [here](protocols/spi) and [here](protocols/i2c).
