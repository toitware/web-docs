# List of drivers

The list of drivers available in the [Toit library](https://libs.toit.io/drivers/library-summary) and on Github can be viewed here:

| _Driver_    | _Description_                     | _Source_                                                                                           | _Example_                                                                             |
| ----------- | --------------------------------- | -------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| `bma2x2`    | Bosch accelerometers              | [link](https://libs.toit.io/drivers/bma2x2/class-Bma2x2)                                           |
| `bme280`    | BME280 environmental sensor       | [link](https://pkg.toit.io/package/github.com%2Ftoitware%2Fbme280-driver@v1.0.0)                   | [Weather station](https://github.com/toitware/bme280_post)                            |
| `bmp280`    | BMP280 environmental sensor       | [link](https://pkg.toit.io/package/bmp280&url=github.com%2Fharshkc03%2Fbmp280-driver&index=latest) |
| `bmg160`    | Gyroscope BMG160                  | [link](https://libs.toit.io/drivers/bmg160/class-Bmg160)                                           |
| `bmm050`    | Magnetometer BMG160               | [link](https://libs.toit.io/drivers/bmm050/class-Bmm050)                                           |
| `color_tft` | IL9xxx and ST77xx screens         | [link](https://github.com/toitware/toit-color-tft)                                                 | [Demo](https://github.com/toitware/toit-color-tft/blob/main/examples/tft_demo.toit)   |
| `MAX31865`  | MAX31865 RTD-to-Digital Converter | [link](https://github.com/toitware/max31865-driver)                                                | [SPI](https://github.com/toitware/max31865-driver/blob/main/examples/main.toit)       |
| `mcp2518fd` | MCP2518FD CAN FD controller       | [link](https://github.com/toitware/mcp2518fd-driver)                                               | [SPI](https://github.com/toitware/mcp2518fd-driver/blob/main/example/spi.toit)        |
| `mcp342x`   | MCP342x ADC devices               | [link](https://github.com/toitware/mcp342x-driver)                                                 | [I2C](https://github.com/toitware/mcp342x-driver/blob/main/examples/main.toit)        |
| `icm20948`  | ICM-20948 9-axis devices          | [link](https://github.com/toitware/icm20948-driver)                                                | [I2C](https://github.com/toitware/icm20948-driver/blob/main/examples/accel_gyro.toit) |
